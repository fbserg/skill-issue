# Publication Checklist

Before creating a public GitHub repo:

- Run a private string scrub for local home paths, account names, client names,
  and credential-related words.
- Confirm any remaining matches are generic code terms or intentionally
  documented examples.
- Run `tools/epic-tools/bin/epic-tools --help`.
- Review `README.md` and `docs/install.md` from a new-user perspective.
- Create the GitHub repo only after the local repo is clean and committed.
